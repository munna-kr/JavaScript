Set (JavaScript)
├── Basics
│   ├── Unique values
│   ├── Insertion order preserved
│   ├── Any type allowed
│   ├── s.size (not length)
│   └── Mutating methods
│       ├── add(value)
│       ├── has(value)
│       ├── delete(value)
│       └── clear()
│
├── Creation
│   ├── new Set()
│   ├── new Set(array)
│   └── Convert to array
│       ├── [...set]
│       └── Array.from(set)
│
├── Iteration
│   ├── for...of
│   ├── forEach()
│   ├── spread `...`
│   └── entries(), values(), keys()
│       └── keys == values (compat w/ Map)
│
├── Core Use Cases
│   ├── Remove duplicates
│   ├── Fast membership check
│   ├── Ensure uniqueness in lists
│   ├── Filtering by allowed set
│   └── Track visited elements
│
├── Set Operations (Manually)
│   ├── Union       (A ∪ B)
│   ├── Intersection (A ∩ B)
│   ├── Difference  (A − B)
│   └── Symmetric diff (A △ B)
│
├── Performance (Big-O)
│   ├── add:     O(1)
│   ├── has:     O(1)
│   ├── delete:  O(1)
│   └── iteration: O(n)
│
├── Objects in Set
│   ├── Unique by reference
│   ├── {a:1} !== {a:1}
│   └── Need Map or key-mapping for structural equality
│
├── Serialization
│   ├── Not JSON serializable directly
│   ├── Convert before JSON.stringify
│   └── Restore: new Set(JSON.parse())
│
├── Advanced Patterns
│   ├── case-insensitive dedupe
│   ├── uniqueness by mapping keys
│   ├── batch removal
│   └── preserve insertion order when deduping
│
└── WeakSet (Advanced cousin)
    ├── Only objects allowed
    ├── Weak references → GC-friendly
    ├── Non-iterable (no size)
    └── Use case: private presence flags
